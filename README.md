## Запуск проекта

### Требования
- Java 21
- Maven 3.8+

### Сторонние библиотеки
JUnit Jupiter: 5.10.1

В системе сборки Maven зависимость выглядит следующим образом:

```xml
<dependency>
    <groupId>org.junit.jupiter</groupId>
    <artifactId>junit-jupiter</artifactId>
    <version>5.10.1</version>
    <scope>test</scope>
</dependency>
```

### Процесс запуска
1. Клонировать репозиторий для локального использования
```bash
    git clone <удалённый репозиторий>
```
2. Собрать проект и упаковать в .jar файл
```bash
    mvn package
```
3. Запустить .jar файл
```bash
    java -jar target/testSHIFT-1.0.jar
```

## Особенности реализации
Регулярные выражения, позволяющие определить число и строку имеют ограничения:
1. Пустые строки игнорируются программой
2. Строки вида "-01", "000002,2" и другие с ведущими нулями будут записаны в исходящий файл как строки
3. Принимаются числа с разделителем дробной части "," и "."
4. Программа закончит выполнение, если на вход не поступит путь или имя .txt файла
5. Если путь входного файла равен одному из путей выходного файла, этот выходной файл будет пропущен, пользователь получит сообщение
6. Если входной файл не найден, он будет пропущен
7. Если возникла проблема с созданием файла под конкретный тип данных, пользователь получит сообщение, программа перейдёт к обработке следующего входного файла
8. Если возникла проблема с записью строки в исходящий файл, пользователь получит сообщение, программа перейдёт к обработке следующего входного файла
9. Программа работает с файловой системой Windows и UNIX-подобных систем
